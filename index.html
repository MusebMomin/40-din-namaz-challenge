About the Challenge

Rules, point system, lifelines, checkpoints, and submission flow

Challenge Goal

  • Build a consistent Salah routine for 40 days.
  • Track daily Salah with one valid submission per date.
  • Protect your streak using lifelines earned through good performance.

Login & Registration

  • Login: Username or Email + Password
  • Register: Full Name + Email + Password
  • Username is auto-derived from the email address.
  • New accounts stay pending until admin approval.

Submission Rules

  • Only today or yesterday can be submitted.
  • Only one submission per entry date is allowed.
  • Each prayer needs exactly one status: Jamat, No Jamat, or Qaza.
  • Takbeer-e-Ula automatically counts as Jamat + Takbeer-e-Ula.
